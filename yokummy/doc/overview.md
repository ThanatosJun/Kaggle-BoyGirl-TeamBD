The multiple datasets below are the datasets used in the first Kaggle competition of this course. The columns in the datasets consist of items from the questionnaire at the beginning of the course.

[boy or girl 2024 train_missingValue.csv] is the training set for this competition, [boy or girl 2024 test no ans_missingValue.csv] is the test set. [Boy_or_girl_test_sandbox_sample_submission.csv[ is an example submission format.

Due to technical issues, both the training and test sets have some missing values.

In this competition, participants need to train a model on the training set and then use the model to predict the gender column in the test set. After submitting the prediction results to the competition page, the score you see is only based on half of the training dataset, and the complete score based on the entire testing dataset will only be revealed after the competition ends.
File descriptions

    boy or girl 2025 train_missingValue.csv - the training set
    boy or girl 2025 test no ans_missingValue.csv - the test set
    Boy_or_girl_test_sandbox_sample_submission.csv - a sample submission file in the correct format

Data fields

    The fields are fairly self-explanatory. The training set and the test set contain the same fields with the target field (gender) in the test set being zero-valued.
